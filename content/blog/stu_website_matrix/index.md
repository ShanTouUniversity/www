ST.U（ShanTou.University）是一个非官方的汕头大学校友社区平台。除了本站（主博客）外，还运营着一系列子站点，共同构成 ST.U 网站矩阵。

## 网站矩阵总览

| 站点 | 域名 | 说明 |
|------|------|------|
| 主站 | [shantou.university](https://shantou.university) | ST.U 主博客，发布各类文章和项目 |
| 导航 | [nav.shantou.university](https://nav.shantou.university) | 汕大常用网站导航，汇集校园服务与工具 |
| 资讯 | [news.shantou.university](https://news.shantou.university) | 非官方校园新闻资讯平台 |
| Voice | [voice.shantou.university](https://voice.shantou.university) | 学生之音，为学生发声 |
| 空间 | [space.shantou.university](https://space.shantou.university) | 博客和创意空间 |

## 各站点简介

### 主站 shantou.university

这是 ST.U 的核心站点，发布关于汕大的各类文章、指南和项目。你可以在这里找到：

- 校园指南
- 学业指导
- 校友故事
- 技术分享

### 导航 nav.shantou.university

汇集了汕大各类常用网站和服务的导航页面。分类包括：

- 办公系统
- 教学单位
- 教学服务
- 校园服务
- 实用工具
- 学习资源
- 校园生活
- 社交娱乐

### 资讯 news.shantou.university

非官方的校园新闻资讯平台，提供中英双语内容。覆盖以下类别：

- 校园动态
- 通知公告
- 学术科研
- 学生活动
- 人物风采
- 媒体聚焦

### Voice voice.shantou.university

"学生之音"，旨在为汕大学生提供一个自由发声的平台。本站使用 Zola 构建，主题简洁专注。

### 空间 space.shantou.university

技术博客和创意空间，分享技术文章与各类有趣项目。包含涂鸦墙等互动功能。

## 开源协作

所有站点的源代码均托管在 [GitHub](https://github.com/ShanTouUniversity) 上，欢迎提交 Issue 或 Pull Request 参与贡献。
